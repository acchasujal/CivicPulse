# Demo Photo Assets Sources

All demo photo assets in this directory were generated via the `generate_image` tool using prompts designed to depict real, amateur smartphone photos of civic infrastructure issues in an Indian urban context (specifically Mumbai).

- `pothole_01.jpg`: Road damage (pothole), daytime, Mumbai urban context.
- `pothole_02.jpg`: Road damage (pothole), daytime, slightly different angle.
- `pothole_03.jpg`: Road damage (pothole), severe depth, daytime.
- `streetlight_01.jpg`: Non-functional streetlight, night/low-light context.
- `streetlight_02.jpg`: Multiple dark streetlights, nighttime context.
- `streetlight_03.jpg`: Non-functional streetlight, low-light view.
